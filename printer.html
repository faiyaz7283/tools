Printer

The _printer is designed to help print data on screen quickly and easily. Just create a new _printer object and start printing with the print() method. There are also many convenient methods to easily customize and style the _printer.

For all the examples in this page, tl is used as a namespace for the tools library. If you are following along with the examples, please make sure to import the latest version of the library in your script first, and assign tl as namespace.

Always create a printer under the barstate.islast condition.

Do not use var when assigning a printer to a variable.

Creating a printer object

// Always create printer under the `barstate.islast` condition. 
if barstate.islast

  // Do not assign with `var`.
  printer = tl._printer.new()

Below we will go through all the available methods and some examples of how to use them. All the examples below assumes we already have a printer object created.

It is recommended to use methods instead of directly interacting with the printer properties. Direct interaction with certain properties will cause unintended behavior.

cell()

cell(): Can be used to change the orientation of arrays to horizontal, or to set a custom cellStyle. The last example assumes we already have a variable called customCellStyle.

// For printing arrays vertically, set horizontal to false (default).
printer.cell(horizontal=false) 

// For printing arrays horizontally, set horizontal to true.
printer.cell(horizontal=true) 

// Set a cellStyle.
printer.cell(cellStyle=customCellStyle)

debug()

debug(): When set to true, adds helpful debugging information to the tooltip.

// Turn on debugger.
printer.debug(true) 

// Turn off debugger (default).
printer.debug(false)

footer()

footer(): Used to set a footer text and footerStyle. The example assumes we already have a variable called customFooterStyle.

// Set footer text and style.
printer.footer("Some footer", footerStyle=customFooterStyle)

gutter()

gutter(): Used to turn gutter on or off, and set the gutterStyle. The last example assumes we already have a variable called custoGutterStyle.

// Set gutter on (default).
printer.gutter(true) 

// Set gutter off.
printer.gutter(false) 

// Set a gutterStyle.
printer.gutter(gutterStyle=customGutterStyle)

header()

header(): Used to set a header text and headerStyle. The example assumes we already have a variable called customHeaderStyle.

// Set header text and style.
printer.header("Some header", headerStyle=customHeaderStyle)

print()

print(): Outputs data onto the screen. Some of the examples below assumes we already have the variables called customTitleStyle and customCellStyle.

// Printing: Basic types.

// string
str  = tl._rndmString()
printer.print(str)

// float
flt  = tl._rndmFloat()
printer.print(flt)

// int
int  = tl._rndmInt()
printer.print(int)

// bool
bool = tl._rndmBool()
printer.print(bool)

// color
clr  = tl._rndmColor()
printer.print(clr)

//-----------------------------------

// Printing: Arrays.

// array<string>
strArr  = tl._rndmStringArray()
printer.print(strArr)

// array<float>
fltArr  = tl._rndmFloatArray()
printer.print(fltArr)

// array<int>
intArr  = tl._rndmIntArray()
printer.print(intArr)

// array<bool>
boolArr = tl._rndmBoolArray()
printer.print(boolArr)

// array<color>
clrArr  = tl._rndmColorArray()
printer.print(clrArr)

//-----------------------------------

// Printing: Matrix.

// matrix<string>
strMtx  = tl._rndmStringMatrix()
printer.print(strMtx)

// matrix<float>
fltMtx  = tl._rndmFloatMatrix()
printer.print(fltMtx)

// matrix<int>
intMtx  = tl._rndmIntMatrix()
printer.print(intMtx)

// matrix<bool>
boolMtx = tl._rndmBoolMatrix()
printer.print(boolMtx)

// matrix<color>
clrMtx  = tl._rndmColorMatrix()
printer.print(clrMtx)

//-----------------------------------

// Printing: Title.

// Add a title.
printer.print(tl._rndmFloatArray, title="Random float array")

// Change the orientation of title.
// Changing orientation for all titles.
printer.title(top=false)
printer.print(tl._rndmFloatArray, title="Random float array")
printer.print(tl._rndmSentence, title="Random sentence")
printer.print(tl._rndmFloatMatrix, title="Random float matrix")

// Changing orientation for single title.
ts = titleStyle.copy(printer.titleStyle)
ts.top := false
printer.print(tl._rndmFloatArray, title="Random float array", titleStyle=ts) // Will be changed.
printer.print(tl._rndmSentence, title="Random sentence") // Back to default orientation.
printer.print(tl._rndmFloatMatrix, title="Random float matrix") // Back to default orientation.

//-----------------------------------

// Printing: Styles.

// Add custom titleStyle and cellStyle to print method.
printer.print(tl._rndmFloatArray, title="Random float array", titleStyle=customTitleStyle) // Custom titleStyle.
printer.print(tl._rndmSentence, title="Random sentence", cellStyle=customCellStyle)  // Custom cellStyle.
printer.print(tl._rndmFloatMatrix, title="Random float matrix", titleStyle=customTitleStyle, cellStyle=customCellStyle) // Both custom styles.
printer.print(tl._rndmIntArray, title="Random integer array") // Back to default styles.

stack()

stack(): Used to change the orientation of the printer. By default, groups data and stacks it vertically. Setting this to false will print each group side by side.

// Turn off stacking.
printer.stack(false) 

// Turn on stacking (default).
printer.stack(true)

theme()

theme(): Used to add a preset style combination to the printer object.

// Available themes:
// blue, gray (default), green, orange, pink, purple, red, white, yellow

// Set 'blue' theme.
printer.theme("blue")

// Set 'gray' theme.
printer.theme("gray")

// Set 'green' theme.
printer.theme("green")

// Set 'orange' theme.
printer.theme("orange")

// Set 'pink' theme.
printer.theme("pink")

// Set 'purple' theme.
printer.theme("purple")

// Set 'red' theme.
printer.theme("red")

// Set 'white' theme.
printer.theme("white")

// Set 'yellow' theme.
printer.theme("yellow")

title()

title(): Used to set the position of titles and titleStyle. The last example assumes we already have a variable called customTitleStyle.

// Place titles on top (default).
printer.title(top=true) 

// Place titles on left.
printer.title(top=false) 

// Set a titleStyle.
printer.title(titleStyle=customtitleStyle)
