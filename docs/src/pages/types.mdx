# Inspect data types.

The [`_type()`](/type#type) is a versatile method that can be used to inspect object type. This method is particularly useful for scripts that rely on 
the type to return a certain value or perform a specific task. The [`_type()`](/type#type) method supports a wide range of data types, including `string`, 
`integer`, `boolean`, `float`, `color`, `table`, `line`, `label`, `box`, `linefill`, `array<type>`, and `matrix<type>`. By leveraging the [`_type()`](/type#type)
method, Pinescript users can create more robust and flexible scripts that can adapt to a wide range of data inputs.

<Note>
  For all the examples in this page, `tools` is used as a namespace for the tools library.
  If you are following along with the examples, please make sure to import the latest 
  version of the library in your script first, and assign `tools` as namespace.
</Note>

<CodeGroup>

```js {{ title: 'Method' }}
// Example testing object types 

varStr = "Hello"
varStr.tools._type() // returns "string".

varFlt = 1.25
varFlt.tools._type() // returns "float".

int varInt = 100
varInt.tools._type() // returns "int".

bool varBool = true
varBool.tools._type() // returns "bool".

color varClr = color.red
varClr.tools._type() // returns "color".

table varTbl = table.new(position, col, row)
varTbl.tools._type() // returns "table".

array<string> varArrS = array.new_string(5,na)
varArrS.tools._type() // returns "array<string>".

matrix<float> varMtxF = matrix.new<float>(4,4,0.5)
varMtxF.tools._type() // returns "matrix<float>".
```
