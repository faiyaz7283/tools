# Styles

A printer object is a highly versatile custom-modified Pinescript table with numerous advanced features. By utilizing 
various categories of style objects, users can further personalize both the printer object and individual print 
statements according to their preferences.

---

## cellStyle

The cellStyle object is similar to styling a table's cell while it targets the [`_printer's`](/_printer#printer) data cells. 
It has the following properties:

### Properties
<Properties>
  <Property name="width" type="float">
    The width of the cell as a % of the indicator's visual space. _Optional. The default is 0_.
  </Property>
  <Property name="height" type="float">
    The height of the cell as a % of the indicator's visual space. _Optional. The default is 3_.
  </Property>
  <Property name="textHalign" type="string">
    The horizontal alignment of the cell's text. Possible values: text.align_left, text.align_center, text.align_right. 
    _Optional. The default is text.align_center_.
  </Property>
  <Property name="textValign" type="string">
    The vertical alignment of the cell's text. Possible values: text.align_top, text.align_center, text.align_bottom. 
    _Optional. The default is text.align_center_.
  </Property>
  <Property name="textColor" type="color">
    The color of the text. _Optional. The default is #FFFFFF_.
  </Property>
  <Property name="textSize" type="string">
    The size of the text. Possible values: size.auto, size.tiny, size.small, size.normal, size.large, size.huge. 
    _Optional. The default is size.auto_.
  </Property>
  <Property name="bgColor" type="color">
    The background color of the text. _Optional. The default is #3C465087_.
  </Property>
  <Property name="textFontFamily" type="string">
    The font family of the text. Possible values: font.family_default, font.family_monospace. _Optional. The default is font.family_default_.
  </Property>
  <Property name="format" type="string">
    Format string targeting footer text. _Optional. The default is na_.
  </Property>
  <Property name="horizontal" type="bool">
   By defaul, array prints vertically (stacked). If horizontal is set to true, array will print horizontally (side to side). 
   This effects **arrays only**. _Optional. The default is false_.
  </Property>
</Properties>

## gutterStyle

The gutterStyle object is similar to styling a table's cell while it targets the [`_printer's`](/_printer#printer) cells between grouped data. 
It has the following properties:

### Properties
<Properties>
  <Property name="gutter" type="bool">
    By default, gutter is active. It targets the cell space between grouped data. _Optional. The default is true_.
  </Property>
  <Property name="width" type="float">
    The width of the cell as a % of the indicator's visual space. _Optional. The default is 0.01_.
  </Property>
  <Property name="height" type="float">
    The height of the cell as a % of the indicator's visual space. _Optional. The default is 0.01_.
  </Property>
</Properties>

## tableStyle

The tableStyle object is similar to styling a table. It has the following properties:

### Properties
<Properties>
  <Property name="bgColor" type="color">
    The background color of the table. _Optional. The default is #000000B3_.
  </Property>
  <Property name="frameColor" type="color">
    The color of the outer frame of the table. _Optional. The default is #3C465047_.
  </Property>
  <Property name="frameWidth" type="int">
    The width of the outer frame of the table. _Optional. The default is 1_.
  </Property>
  <Property name="borderColor" type="color">
    The color of the borders of the cells (excluding the outer frame). _Optional. The default is na_.
  </Property>
  <Property name="borderWidth" type="int">
    The width of the borders of the cells (excluding the outer frame). _Optional. The default is 2_.
  </Property>
</Properties>
