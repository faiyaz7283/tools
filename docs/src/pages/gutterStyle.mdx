# gutterStyle

The gutterStyle object is similar to styling a table's cell while it targets the [`_printer's`](/_printer) cells between grouped data. 
It has the following properties:

### Properties
<Properties>
  <Property name="gutter" type="bool">
    By default, gutter is active. It targets the cell space between grouped data. _Optional. The default is true_.
  </Property>
  <Property name="width" type="float">
    The width of the cell as a % of the indicator's visual space. _Optional. The default is 0.01_.
  </Property>
  <Property name="height" type="float">
    The height of the cell as a % of the indicator's visual space. _Optional. The default is 0.01_.
  </Property>
</Properties>
