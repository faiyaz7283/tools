# Join method.
This method is used to connect array items into a single string using a separator.

---

## _join()
Join an array to a string using a separator.
<Properties>
  <Property name="this" type="object">
    Any object of type array\<bool\>, array\<string\>, array\<float\>, array\<int\> or array\<color\>.
  </Property>
  <Property name="sep" type="string">
    A string separator. _Optional. The default is , (pipe)_.
  </Property>
  <Property name="returns" type="string">
    A joined string.
  </Property>
</Properties>

---

# Split methods.
There are three types of split methods - regular split, split pair, and split shift.

## _commaSplit()
Split a comma separated string into an array.
<Properties>
  <Property name="this" type="object">
    Any object of type string.
  </Property>
  <Property name="returns" type="array<string>">
    A string array.
  </Property>
</Properties>

<Properties>

    Any object of type string.
  </Property>

  </Property>
  <Property name="returns" type="string">
    A string.
  </Property>

## _commaSplitPair()
