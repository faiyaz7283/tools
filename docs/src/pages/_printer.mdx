# Printer
Let's see how the _printer object structure, and all its availble methods.

## _printer

The _printer object is the core for the print method. It has the following properties:

<Note>
  Please do not assign any value to properties that starts with underscore. They are meant for internal use only.
  For all other properties, please use dedicated getter and setter methods to interact with them. 
</Note>

### Properties
<Properties>
  <Property name="header" type="string">
    A header for the _printer content. _Optional. The default is na_.
  </Property>
  <Property name="footer" type="string">
    A header for the _printer content. _Optional. The default is na_.
  </Property>
  <Property name="stack" type="bool">
    If true, each new grouped data are stacked under, if false, they are placed on right side. _Optional. The default is true_.
  </Property>
  <Property name="loc" type="string">
    The screen location to place the printer object. This is same as pinescripts table positions. Possible values are: 
    position.top_left, position.top_center, position.top_right, position.middle_left, position.middle_center, position.middle_right, 
    position.bottom_left, position.bottom_center, position.bottom_right. _Optional. The default is position.middle_center_.
  </Property>
  <Property name="debug" type="bool">
    If true, then print method will provide some basic debug info. _Optional. The default is false_.
  </Property>
  <Property name="tableStyle" type="tableStyle">
    A tableStyle object. _Optional. The default is na_.
  </Property>
  <Property name="headerStyle" type="headerStyle">
    A headerStyle object. _Optional. The default is na_.
  </Property>
  <Property name="footerStyle" type="footerStyle">
    A footerStyle object. _Optional. The default is na_.
  </Property>
  <Property name="titleStyle" type="titleStyle">
    A titleStyle object. _Optional. The default is na_.
  </Property>
  <Property name="cellStyle" type="cellStyle">
    A cellStyle object. _Optional. The default is na_.
  </Property>
  <Property name="gutterStyle" type="gutterStyle">
    A gutterStyle object. _Optional. The default is na_.
  </Property>
  <Property name="_table" type="table">
    A Table object. _Optional. The default is na_.
  </Property>
  <Property name="_cTotal" type="int">
    Column total. _Optional. The default is 100_.
  </Property>
  <Property name="_rTotal" type="int">
    Row total. _Optional. The default is 100_.
  </Property>
  <Property name="_slot" type="string">
    A comma separated string containing column and row number of last active cell used. _Optional. The default is na_.
  </Property>
  <Property name="_maxCol" type="int">
    Size counter of maximum column used. _Optional. The default is 0_.
  </Property>
  <Property name="_init" type="bool">
    A checker to see if _printer object has been initialized. _Optional. The default is false_.
  </Property>
</Properties>
