# Join and split
The tools library provides some custom array and string methods that expand Pinescript's own `array.join()` and `str.split()` methods.

<Note>
  For all the examples in this page, `tools` is used as a namespace for the tools library.
  If you are following along with the examples, please make sure to import the latest 
  version of the library in your script first, and assign `tools` as namespace.
</Note>

## Join
The [`_join()`](/join-splits#join) method is similar to the `array.join()` method in Pinescript, but it extends this functionality to boolean and color arrays.

<CodeGroup>

```js {{ title: 'Method' }}
// Joining array<string>

strArray = array.from("apple", "banana", "carrot")
strArray.tools._join()          // returns the string "apple,banana,carrot".
strArray.tools._join(sep=" ")   // returns the string "apple banana carrot".
strArray.tools._join(sep="-")   // returns the string "apple-banana-carrot".

// Joining array<float>

fltArray = array.from(1.2, 2.1, 3.098, 4.0089)
fltArray.tools._join()          // returns the string "1.2,2.1,3.098,4.0089".
fltArray.tools._join(sep=" ")   // returns the string "1.2 2.1 3.098 4.0089".
fltArray.tools._join(sep="-")   // returns the string "1.2-2.1-3.098-4.0089".

// Joining array<int>

intArray = array.from(1, 2, 3, 4, 5)
intArray.tools._join()          // returns the string "1,2,3,4,5".
intArray.tools._join(sep=" ")   // returns the string "1 2 3 4 5".
intArray.tools._join(sep="-")   // returns the string "1-2-3-4-5".

// Joining array<bool>

boolArray = array.from(true, false, false, true, false)
boolArray.tools._join()         // returns the string "true,false,false,true,false".
boolArray.tools._join(sep=" ")  // returns the string "true false false true false".
boolArray.tools._join(sep="-")  // returns the string "true-false-false-true-false".

// Joining array<color>

clrArray = array.from(color.red, color.black, color.white)
clrArray.tools._join()          // returns the string "255|82|82|0,54|58|69|0,255|255|255|0".
clrArray.tools._join(sep=" ")   // returns the string "255|82|82|0 54|58|69|0 255|255|255|0".
clrArray.tools._join(sep="-")   // returns the string "255|82|82|0-54|58|69|0-255|255|255|0".
```

```js {{ title: 'Function' }}

// Joining array<float>

// Joining array<int>

tools._join(intArray, sep=" ")  // returns the string "1 2 3 4 5".

// Joining array<bool>
```

## Split

The tools library provides several convenient split methods that work similarly to the `str.split()` method in Pinescript. 
These methods are named with their separator (comma, pipe or colon) and follow a naming convention: 

- Split into an array.
  - [`_commaSplit()`](/join-splits#comma-split)
  - [`_pipeSplit()`](/join-splits#pipe-split)
  - [`_colonSplit()`](/join-splits#colon-split)
- Split and get just a single item.
  - [`_commaSplit()`](/join-splits#comma-split-2)
  - [`_pipeSplit()`](/join-splits#pipe-split-2)
  - [`_colonSplit()`](/join-splits#colon-split-2) 
- Split into a single tuple pair of first two items.
  - [`_commaSplitPair()`](/join-splits#comma-split-pair)
  - [`_pipeSplitPair()`](/join-splits#pipe-split-pair)
  - [`_colonSplitPair()`](/join-splits#colon-split-pair)
- Split into an array without the first item.
  - [`_commaSplitShift()`](/join-splits#comma-split-shift)
  - [`_pipeSplitShift()`](/join-splits#pipe-split-shift)
  - [`_colonSplitShift()`](/join-splits#colon-split-shift)

<CodeGroup>

```js {{ title: 'Method' }}
// Separator: Comma

commaStr = "apple,banana,carrot"
commaStr.tools._commaSplit()        // returns an array<string>("apple", "banana", "carrot").
commaStr.tools._commaSplit(idx=2)   // returns the string "carrot".
commaStr.tools._commaSplitPair()    // returns a tuple containing ["apple", "banana"].
commaStr.tools._commaSplitShift()   // returns an array<string>("banana", "carrot") without "apple".

// Separator: Pipe

pipeStr = "apple|banana|carrot"
pipeStr.tools._pipeSplit()          // returns an array<string>("apple", "banana", "carrot").
pipeStr.tools._pipeSplit(idx=2)     // returns the string "carrot".
pipeStr.tools._pipeSplitPair()      // returns a tuple containing ["apple", "banana"].
pipeStr.tools._pipeSplitShift()     // returns an array<string>("banana", "carrot") without "apple".

// Separator: Colon

colonStr = "apple:banana:carrot"
colonStr.tools._colonSplit()        // returns an array<string>("apple", "banana", "carrot").
colonStr.tools._colonSplit(idx=2)   // returns the string "carrot".
colonStr.tools._colonSplitPair()    // returns a tuple containing ["apple", "banana"].
colonStr.tools._colonSplitShift()   // returns an array<string>("banana", "carrot") without "apple".
```

