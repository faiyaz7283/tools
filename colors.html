Color

There are two ways to generate colors using the tools library. One, using string method with constant color names. Two, using a gradient object method to get color based on number value.

For all the examples in this page, tools is used as a namespace for the tools library. If you are following along with the examples, please make sure to import the latest version of the library in your script first, and assign tools as namespace.

Constants

Using the _color() method with constant color names. These constants follow a naming convention of color name followed by a type of shade: <color>_bright, <color>_light, <color>_medium, and <color>_dark. For example, "aqua_bright"._color() returns the color #00FFFF.

The color method also accepts a parameter for passing transparency value. By default, it's always 0 (full transparency). For example, "aqua_bright"._color(transp=50) returns the color #00FFFF7F.

// All available preset colors.

"aqua_bright".tools._color()   // returns the color #00FFFF.
"aqua_light".tools._color()    // returns the color #00C8FF.
"aqua_medium".tools._color()   // returns the color #0096FF.
"aqua_dark".tools._color()     // returns the color #0078FF.

"black_dark".tools._color()    // returns the color #000000.

"orange_bright".tools._color() // returns the color #FF6400.

Gradient

The second use of the _color() method is with the _gradient object. Let's look at some examples below.

Usage example:

// Create a new _gradient object.
var gradient = tools._gradient.new(
                 max=100, 
                 min=-100, 
                 mid=0, 
                 up=color.lime, 
                 neutral=color.white, 
                 down=color.red)

// Examples of getting gradient colors.
gradient.tools._color(num=25.08)   // returns color #BFF8DC.
gradient.tools._color(num=-25.90)  // returns color #FFD2D2.
gradient.tools._color(num=0)       // returns color #FFFFFF.
gradient.tools._color(num=100)     // returns color #00E676.
gradient.tools._color(num=-100)    // returns color #FF5252.
gradient.tools._color(num=62.35)   // returns color #60EFA9.
