# tableStyle

The tableStyle object is similar to styling a table. It has the following properties:

### Properties
<Properties>
  <Property name="bgColor" type="color">
    The background color of the table. _Optional. The default is #000000B3_.
  </Property>
  <Property name="frameColor" type="color">
    The color of the outer frame of the table. _Optional. The default is #3C465047_.
  </Property>
  <Property name="frameWidth" type="int">
    The width of the outer frame of the table. _Optional. The default is 1_.
  </Property>
  <Property name="borderColor" type="color">
    The color of the borders of the cells (excluding the outer frame). _Optional. The default is na_.
  </Property>
  <Property name="borderWidth" type="int">
    The width of the borders of the cells (excluding the outer frame). _Optional. The default is 2_.
  </Property>
</Properties>
