# Generators

The tools library provides a set of custom data generator functions. These generator functions
can be used to produce a single data or multiple data across all pinescripts basic data types - 
string, float, integer (including unix timestamp), boolean, color. These data can be generated 
as single entity or using pinescript's array or matrix.

<Note>
  For all the examples in this page, `tools` is used as a namespace for the tools library.
  If you are following along with the examples, please make sure to import the latest 
  version of the library in your script first, and assign `tools` as namespace.
</Note>

## Strings
To generate random string(s), use the following functions:

- [`_rndmChar()`](/string-generators#rndm-char)
- [`_rndmString()`](/string-generators#rndm-string)
- [`_rndmWord()`](/string-generators#rndm-word)
- [`_rndmSentence()`](/string-generators#rndm-sentence)
- [`_rndmStringArray()`](/string-generators#rndm-string-array)
- [`_rndmCharArray()`](/string-generators#rndm-char-array)
- [`_rndmStringMatrix()`](/string-generators#rndm-string-matrix)
- [`_rndmCharMatrix()`](/string-generators#rndm-char-matrix)

<CodeGroup title="Generating random string(s).">

  ```js {{ title: 'Random character' }}
  tools._rndmChar()
  // returns a random a-z character.

  tools._rndmCharArray()        
  // returns random size array<string>, each filled with a single random a-z character.

  tools._rndmCharArray(size=5)     
  // returns array<string> size 5, each filled with a single random a-z character.         
  
  tools._rndmCharMatrix()   
  // returns matrix<string> with random rows and columns, each filled with single random a-z character.

  tools._rndmCharMatrix(rows=3, columns=4)  
  // returns 3x4 matrix<string>, each filled with a single random a-z character.
  ```

  ```js {{ title: 'Random word' }}
  tools._rndmWord()
  // returns a random word.

  tools._rndmWord(len=3)                    
  // returns a random word with length of 3 characters.
  ```

  ```js {{ title: 'Random sentence' }}
  tools._rndmSentence()
  // returns a random sentence.

  tools._rndmSentence(size=4) 
  // returns a random sentence with a size of 4 words.                
  ```

  ```js {{ title: 'Random string' }}
  tools._rndmString()                         
  // returns a random string.

  tools._rndmString(len=8)                   
  // returns a random string with length of 8 characters.

  tools._rndmStringArray()                    
  // returns random size array<string>, each filled with random length strings.

  tools._rndmStringArray(size=4, len=3) 
  // returns array<string> size 4, each filled with fixed 3 character length string.

  tools._rndmStringMatrix()                   
  // returns matrix<string> with random rows and columns, each filled with random length strings.

  tools._rndmStringMatrix(rows=2, columns=2)  
  // returns 2x2 matrix<string>, each filled with random length strings.
  ```

</CodeGroup>

## Floats
To generate random float(s), use the following functions:

- [`_rndmFloat()`](/float-generators#rndm-float)
- [`_rndmFloatArray()`](/float-generators#rndm-float-array)
- [`_rndmFloatMatrix()`](/float-generators#rndm-float-matrix)

```js {{ title: 'Generating random float(s)' }}
tools._rndmFloat()                                                    
// returns 66.17.

tools._rndmFloat(min=25, max=30, precision=5)                         
// returns 25.01369.

tools._rndmFloatArray()                                               
// returns random size array<float>, each filled with random floats in range 0-100 with a 
// precision of 2 decimal points.

tools._rndmFloatArray(size=3, min=0, max=1, precision=8)              
// returns array<float> size 3, each filled with random floats in range 0-1 with a precision 
// of 8 decimal points.

tools._rndmFloatMatrix()                                              
// returns matrix<float> with random rows and columns, each filled with random floats in range 
// 0-100 with a precision of 2 decimal points.

tools._rndmFloatMatrix(rows=4, columns=4, min=1, max=5, precision=1)  
// returns matrix<float> with 4 by 4 rows and columns, each filled with random floats in range 
// 1-5 with a precision of 1 decimal points.
```

## Integers
To generate random integer(s), use the following functions:

- [`_rndmInt()`](/int-generators#rndm-int)
- [`_rndmIntArray()`](/int-generators#rndm-int-array)
- [`_rndmIntMatrix()`](/int-generators#rndm-int-matrix)

```js {{ title: 'Generating random integer(s)' }}
tools._rndmInt()                                      
// returns 4.

tools._rndmInt(min=25, max=30)                        
// returns 27.

tools._rndmIntArray()                                 
// returns random size array<int>, each filled with random integer in range 1-100.

tools._rndmIntArray(size=3, min=-5, max=5)            
// returns array<int> size 3, each filled with random integer in range -5-5.

tools._rndmIntMatrix()                                
// returns matrix<int> with random rows and columns, each filled with random integers in 
// range 1-100.

tools._rndmIntMatrix(rows=4, columns=4, min=1, max=5) 
// returns 4x4 matrix<int>, each filled with random integers in range 1-5.
```

## Colors
To generate random color(s), use the following functions:

- [`_rndmColor()`](/color-generators#rndm-color)
- [`_rndmColorArray()`](/color-generators#rndm-color-array)
- [`_rndmColorMatrix()`](/color-generators#rndm-color-matrix)

```js {{ title: 'Generating random color(s)' }}
tools._rndmColor()                                    
// returns color #D7AD84.

tools._rndmColor(trnsp=true)                          
// returns color #C0D88E3B.

tools._rndmColorArray()                               
// returns random size array<color>, each filled with random color value.

tools._rndmColorArray(size=2, trnsp=true)             
// returns array<color> size 2, each filled with random color value and random transparency.

tools._rndmColorMatrix()                              
// returns matrix<color> with random rows and columns, each filled with random color value and 
// random transparency.

tools._rndmColorMatrix(rows=3, columns=3, trnsp=true) 
// returns 3x3 matrix<color>, each filled with random color value and random transparency.
```

## unix timestamps
To generate random unix timestamp(s), use the following functions:

- [`_rndmTimestamp()`](/unix-timestamp-generators#rndm-timestamp)
- [`_rndmTimestampArray()`](/unix-timestamp-generators#rndm-timestamp-array)
- [`_rndmTimestampMatrix()`](/unix-timestamp-generators#rndm-timestamp-matrix)

```js {{ title: 'Generating random unix timestamp(s)' }}
tools._rndmTimestamp()                                    
// returns 303740247000.

tools._rndmTimestamp(year=2016)                           
// returns 1470434775000.

tools._rndmTimestamp(year=2002, month=10)                 
// returns 1477449509000.

tools._rndmTimestamp(year=1998, month=02, day=12)         
// returns 887328265000.

tools._rndmTimestamp(day=12, hour=12, minute=30)          
// returns 963419451000.

tools._rndmTimestampArray()                               
// returns random size array<int>, each filled with random unix timestamp.

tools._rndmTimestampArray(size=2, year=2000, month=01)    
// returns array<int> size 2, each filled with random unix timestamp with fixed year 2000 
// and January month.

tools._rndmTimestampMatrix()                              
// returns matrix<int> with random rows and columns, each filled with random unix timestamp.

tools._rndmTimestampMatrix(rows=2, columns=2, year=1999)  
// returns 2x2 matrix<int>, each filled with random unix timestamp with fixed year 1999.
```
