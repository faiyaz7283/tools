# Float generators.
A set of functions to help with generating random types of [`float`](https://www.tradingview.com/pine-script-reference/v5/#op_float). 
Generators can return a single value or grouped values using arrays and matrix. 

---

## _rndmFloat()
Generate a random float number.
<Properties>
  <Property name="min" type="float">
    Minimum value allowed. _Optional. The default is 0_.
  </Property>
  <Property name="max" type="float">
    Maximum value allowed. _Optional. The default is 100_.
  </Property>
  <Property name="precision" type="int">
    Decimal places to which the number will be rounded. _Optional. The default is 2_.
  </Property>
  <Property name="returns" type="float">
    A random float number.
  </Property>
</Properties>

---

## _rndmFloatArray()
Generate a random float array.
<Properties>
  <Property name="size" type="int">
    The size of the array. By default its set to 0, which implies a random length between 3-10 values. _Optional. The default is 0_.
  </Property>
  <Property name="min" type="float">
    Minimum value allowed. _Optional. The default is 0_.
  </Property>
  <Property name="max" type="float">
    Maximum value allowed. _Optional. The default is 100_.
  </Property>
  <Property name="precision" type="int">
    Decimal places to which the number will be rounded. _Optional. The default is 2_.
  </Property>
  <Property name="returns" type="array<float>">
    A random float array.
  </Property>
</Properties>

---

## _rndmFloatMatrix()
Generate a random float matrix.
<Properties>
  <Property name="rows" type="int">
    Total row size. By default its set to 0, which implies a random length between 3-10 values. _Optional. The default is 0_.
  </Property>
  <Property name="columns" type="int">
    Total column size. By default its set to 0, which implies a random length between 3-10 values. _Optional. The default is 0_.
  </Property>
  <Property name="min" type="float">
    Minimum value allowed. _Optional. The default is 0_.
  </Property>
  <Property name="max" type="float">
    Maximum value allowed. _Optional. The default is 100_.
  </Property>
  <Property name="precision" type="int">
    Decimal places to which the number will be rounded. _Optional. The default is 2_.
  </Property>
  <Property name="returns" type="matrix<float>">
    A random float matrix.
  </Property>
</Properties>
