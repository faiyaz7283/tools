# Color
There are two ways to generate colors using the tools library. One, using string method with
constant color names. Two, using a gradient object method to get color based on number value.

<Note>
  For all the examples in this page, `tl` is used as a namespace for the tools library.
  If you are following along with the examples, please make sure to import the latest 
  version of the library in your script first, and assign `tl` as namespace.
</Note>

## Constants
Using the [`_color()`](#) method with constant color names. These constants follow a naming 
convention of color name followed by a type of shade: &lt;color&gt;_bright, &lt;color&gt;_light, &lt;color&gt;_medium, and &lt;color&gt;_dark. 
For example, `"aqua_bright"._color()` returns the color #00FFFF.

The color method also accepts a parameter for passing transparency value. By default, it's always 0 (full transparency). For example, 
`"aqua_bright"._color(transp=50)` returns the color #00FFFF7F.

## Gradient
The second use of the [`_color()`](#) method is with the [`_gradient`](/_gradient) object. 
Let's look at some examples below.

### Usage example:

<CodeGroup>

```js {{ title: 'Method' }}
// Create a new _gradient object.
var gradient = tl._gradient.new(
                 max=100, 
                 min=-100, 
                 mid=0, 
                 up=color.lime, 
                 neutral=color.white, 
                 down=color.red)

// Examples of getting gradient colors.
gradient.tl._color(num=25.08)   // returns color #BFF8DC
gradient.tl._color(num=-25.90)  // returns color #FFD2D2
gradient.tl._color(num=0)       // returns color #FFFFFF
gradient.tl._color(num=100)     // returns color #00E676
gradient.tl._color(num=-100)    // returns color #FF5252
gradient.tl._color(num=62.35)   // returns color #60EFA9
```

```js {{ title: 'Function' }}
// Create a new _gradient object.
var gradient = tl._gradient.new(
                 max=100, 
                 min=-100, 
                 mid=0, 
                 up=color.lime, 
                 neutral=color.white, 
                 down=color.red)

// Examples of getting gradient colors.
tl._color(gradient, num=25.08)  // returns color #BFF8DC
tl._color(gradient, num=-25.90) // returns color #FFD2D2
tl._color(gradient, num=0)      // returns color #FFFFFF
tl._color(gradient, num=100)    // returns color #00E676
tl._color(gradient, num=-100)   // returns color #FF5252
tl._color(gradient, num=62.35)  // returns color #60EFA9
```
