# Casting methods.
A set of methods to help with casting for the following object types.

- [`boolean`](https://www.tradingview.com/pine-script-reference/v5/#op_bool)
- [`float`](https://www.tradingview.com/pine-script-reference/v5/#op_float)
- [`integer`](https://www.tradingview.com/pine-script-reference/v5/#op_int)
- [`color`](https://www.tradingview.com/pine-script-reference/v5/#op_color)
- [`string`](https://www.tradingview.com/pine-script-reference/v5/#op_string)


---
